== Description ==
**Accepts one time payments and run transactions so that users join as members.**

Once activated and set up, this addon enables customers to purchase subscriptions by making a single payment.

**This addon is an extension of [Ultimate Membership Pro](https://ultimatemembershippro.com) plugin.**

== Installation ==
1. Download the plugin (.zip file).
2. In WordPress Dashboard go to Plugins > Add New.
3. Upload Plugin by pressing the upload button.
4. Activate the plugin.
5. In [Ultimate Membership Pro](https://ultimatemembershippro.com) plugin dashboard go to navigation menu and find EXTENSIONS tab. Upon clicking the menuâ€™s item, a box with Razorpay module will be available.


For more details, visit our [documentation](https://store.wpindeed.com/documentation/razorpay-payment-gateway/) page.
